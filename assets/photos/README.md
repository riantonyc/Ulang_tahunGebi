# 📸 Foto Blandina Gebi

Letakkan foto Blandina Gebi di folder **`local/`** (bukan di folder ini).

**Petunjuk:**
1. Letakkan foto di folder `local/` dengan nama `blandina.jpg` (atau `.png`)
2. Format yang didukung: `.jpg`, `.jpeg`, `.png`, `.webp`
3. Ukuran ideal: minimal 400x400 pixel, rasio 1:1 paling ideal
4. Foto akan diproses & di-resize otomatis saat build

Folder ini (`assets/photos/`) disediakan untuk dokumentasi saja.
